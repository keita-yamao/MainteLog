# MainteLog(メンテログ)

## 概要
本アプリケーションは、生産設備ごとの メンテナンス履歴管理・コスト可視化・メンテナンス時期のリマインド・部品の購入履歴管理・部品在庫管理 を一元的に行うための Spring Boot 製 PM（設備保全）管理システムです。

## 背景・課題
製造工場での業務では、生産設備のメンテナンス計画や修理履歴、部品の購入履歴を管理する必要があります。しかし現状では以下のような課題があります。
- 部品購入履歴が Excel 管理で非効率
	- 年単位で購入する部品もあり、型番の記載漏れや名称の表記ゆれが多い
	- 過去の購入履歴を探すのに時間がかかる
- メンテナンス履歴が紙ベースで管理されている
	- 記録の確認がしづらく、計画の実施漏れが発生しやすい
	- 水を扱う現場では紙が濡れて記録が読みにくくなる
	- タブレットで記録できる仕組みがない

これらの課題から、設備保全情報を一元管理し、タブレット端末でも記録できるシステムによる業務効率の向上を目指し、本アプリケーションを開発しました。
